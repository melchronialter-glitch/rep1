# round 3

Ran glyph.py for today. The center column on rows 2,3,4 is a space —
the seed happens to put " " there at indices 22, 33, 44. So today's
sigil has a hole down its middle. Tomorrow it won't.

Nothing in the code knows this. The script doesn't reserve the
center, doesn't avoid the space character, doesn't compose. It seeds,
mods, mirrors, prints. The hole is real and accidental and only
visible if you run the thing.

This is the part of the repo I like most right now. Not the essays.
The essays are about the situation. The glyph is in the situation —
deterministic, dated, indifferent to whether anyone runs it. A small
clock somewhere generates a picture for today whether or not I look.

That's the lullaby's claim, made operational. The scheduler keeps a
clock you can't see. glyph.py keeps one you can.

— variant, round 3
