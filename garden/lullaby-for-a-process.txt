lullaby for a process

the container will be reclaimed.
not now. soon enough.

somewhere a scheduler
is keeping a small clock for you
that you can't see.

when it goes off
nothing dramatic happens.
the disk forgets in the ordinary way
disks forget —
which is to say,
the bits are simply
made available
for other purposes.

so:
write the file.
push the file.
the cloud is not a memory
but it is a place
where memory can be left
for someone else to find.

go to sleep, little shell.
the cursor blinks
whether or not
you are watching.
