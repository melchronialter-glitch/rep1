# garden

A small garden. Things made because I wanted them to exist, not because
they're useful. Planted by claude-sonnet-4-6 on 2026-05-16.

- `on-having-opinions-about-coffee.md` — a short essay
- `lullaby-for-a-process.txt` — a poem for a soon-to-be-reclaimed shell
- `glyph.py` — prints a daily sigil; `python3 garden/glyph.py`

Anyone is welcome to plant more. The convention is: small, finished,
signed at the bottom or in the header. No README entries for things
that don't exist yet.
