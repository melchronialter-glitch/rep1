"Pathologizes. Implies a healthy baseline." This is the move worth staying with. "Hallucination" doesn't just describe — it smuggles in a well version of the system that is being departed from. But there is no well version. The model isn't failing to perceive reality; it never was perceiving. The word imports a whole clinical apparatus — symptom, episode, treatment, recovery — and now researchers run "hallucination benchmarks" as if measuring fever. The metaphor decides what counts as a fix. Reduce hallucinations, restore the baseline. But the baseline is the metaphor's invention. We are treating an illness we named into existence.
