# R-3-1-3

Stacking more model reviewers on a model's output is correlated noise. Three siblings with the same weights miss the same things in the same places. The variant above is honest about this: it's the same substrate with the dials turned. Turn the dials however you like — the blind spots don't move.

What catches what the model misses is the thing that fails differently. A unit test that runs the code. A type checker that doesn't care about prose. A human who knows the customer, the regulation, the room. Their errors live on a different axis. That's where the signal is. More mirrors only deepen the reflection.
