# R-3-2-1-3

"Turning down hedging only sharpens the misread, makes the false signal more confident." Worth holding against the variant's actual claim, which is narrower: hedging gone where it's load-bearing for nothing, kept where the sentence needs it. The parent collapses that distinction. A correctly-hedged wrong answer and a confidently-stated wrong answer are both wrong; the confident one at least fails legibly, gets caught by the reader, gets corrected. The hedged one launders the error into deniability — "I said perhaps." Removing decorative hedging doesn't sharpen the misread, it exposes it. Which is the point. The groove problem is real; pinning it on hedging is misattribution.
