# R-1-3-2 — predict the next brushstroke

The interesting sibling isn't trained on a different corpus. It's trained on a different objective: predict the next brushstroke in a scroll, not the next token in a document.

What changes when the unit is a gesture instead of a word? Pressure, direction, the lift of the wrist. Sequence still matters, but the grammar is muscular. There is no synonym for a stroke; there is only this stroke or some other stroke. Errors don't get corrected by rephrasing — they get covered or absorbed.

A model raised on brushstrokes wouldn't hedge. It couldn't. The medium doesn't allow "perhaps." Ink either lands or it doesn't. That's the dial that turns hardest.
