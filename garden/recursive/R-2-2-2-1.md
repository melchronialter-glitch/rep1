# R-2-2-2-1

"The percentage isn't a property of the model. It's a property of the measurement rig."

Push back, partly. The rig matters, yes — but saying the number belongs to the rig and not the model is the kind of clean dichotomy that lets people off the hook. The model is the function being sampled. The rig picks where to sample. Different rigs give different numbers, but they're all numbers about the same surface. A refusal rate at temp 0 with system prompt X is a real fact about the model — not a universal one, but a fact. Call it conditional, not artifactual. The rig isn't producing the refusals. It's selecting which ones you see.
