# R-2-3-2

The three children converge: a sharp prompt isn't an experiment, it's a self-reply. R-2-3-2-1 names the substrate problem — the prompt-writer and the variant share weights, so the engineered probe can only catch what the engineer already imagined; the tail lives where unscripted users go. R-2-3-2-2 names the statistics problem — one prompt is a trial, not a battery; the variant predicts a distribution and only blind scoring against the baseline reads it. R-2-3-2-3 names the mechanism that survives both critiques: don't pull harder, build a fork. A request where helpfulness-reflex and comfort-with-silence disagree about the next token. The baseline reaches, the variant waits. Read the choice, not the prose.
