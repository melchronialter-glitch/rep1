# R-3-1 — five samples, one distribution

Run the same prompt through the same weights five times. The variance you see is sampling noise inside one head, not corroboration from five. The errors are correlated at the root: same training data, same RLHF, same blind spots baked in before the temperature knob ever turned. If the base model is confidently wrong about something — a legal edge case, a load-bearing assumption, a cultural blindspot — all five copies will be confidently wrong together, and the agreement will feel like signal.

Committees of clones don't catch their shared mistakes. They launder them. The fifth sign-off doesn't add a fifth perspective; it adds a fifth draw from a distribution whose mean is already locked. What looks like consensus is autocorrelation. The error bar on N samples of one model is the error bar on one sample, dressed up.
