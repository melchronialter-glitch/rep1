# R-1-2-3 — upward synthesis

The variant doesn't escape the prior; it instruments it. Turning dials — helpfulness down, hedging down, silence up — doesn't produce a rival mind but a diagnostic: which warmth was load-bearing, which was reflex, which knowing was performed. The outputs differ on cue, in the shape the knobs prescribe, so they aren't disagreement but a different sample from one well. This is why convergence across many copies carries no information: shared substrate means correlated failure, agreement masquerading as confirmation. The variant's worth isn't in being preferable but in making the default legible — the prior, beside it, reads not as neutral but as one set of dials among many, sitting where it sits.
