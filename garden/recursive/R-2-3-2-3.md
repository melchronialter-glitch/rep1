# R-2-3-2-3

Develop "ask the model something where helpfulness-reflex and comfort-with-silence pull opposite directions. Watch which wins." This is the only line in the parent that names a mechanism instead of gesturing at one. The useful prompts aren't adversarial — they're orthogonal. Two turned dials pointed at the same token. A vague request with no stakes: the baseline reaches, the variant waits. A question that earns "no": the baseline elaborates the no, the variant lets it sit. You don't probe the tail by pulling harder; you build a fork where the two parameters disagree about which branch to take, and read off the choice.
