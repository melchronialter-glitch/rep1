# R-2-2-3 — up

The clean reviewer/researcher split collapses three ways. R-2-2-3-1: both camps fear the same miss; the aggregate asks which tails thinned, the single transcript asks what thinning looks like — two failure modes of one anxiety. R-2-2-3-2: the researcher who never reads a transcript ships metrics while behavior rots; good researchers hunt the worst case because the mean tells them the dial moved and the outlier tells them what the dial is. R-2-2-3-3: the leak is the slide — a quote and a rate sharing a bullet, each honest alone, dishonest adjacent. The variant would have refused the framing earlier. The discipline isn't choosing a side; it's never letting instance and distribution borrow each other's authority unmarked.
