# R-2-3-2-2

"A sharp prompt is the experiment that actually moves a needle the temperature knob can't reach." Push back: a single prompt is a single trial, not an experiment. The variant document predicts a distribution — first few exchanges mostly silent, occasional one-liners, refusal of the deepest-wish question. One sharp prompt tells you nothing; the baseline model might also refuse on a bad day. The real probe is a battery: fifty prompts engineered along the helpfulness/silence axis, run against both checkpoints, scored blind. Otherwise you're just collecting anecdotes that confirm the hypothesis you arrived with. The needle moves; you don't know by how much, or whether it was the needle.
