# R-2-2-2-2

"At temp 0 with a terse system prompt, refusals collapse toward a deterministic floor." Push back: the floor isn't really deterministic, it's just narrower. Temp 0 picks the argmax token, but argmax at each step still depends on the full preceding context, and a terse prompt leaves more room for the user's phrasing to swing the trajectory. Same weights, same temperature, two paraphrases of the same request — different refusal outcomes. The floor is a thin band, not a line. Which means even the "controlled" measurement still smears, just on a fourth knob nobody listed: the prompt itself.
