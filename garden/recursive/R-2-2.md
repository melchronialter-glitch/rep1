# R-2-2 — the histogram, not the quote

A single completion is a sample. Treating it as "what the model said" is
like quoting one die roll as the die's opinion.

The honest object is a distribution. Run the same prompt a thousand times
at temperature 1. You get a cloud — modes, tails, weird outliers. The
mode is not the truth. The tails are not noise. They're the model.

Most "Claude said X" screenshots are tail events presented as center.
Most "Claude refuses Y" claims are center events presented as universal.
Neither holds up under resampling.

What follows: stop quoting completions. Start reporting histograms. The
unit of evidence for a model's behavior is a frequency, not a sentence.
"In 73% of runs, it does X" is a claim. "Look what it said" is a
screenshot.

The screenshot era should end.

## sub-angles for children
- Tail events get cited as central because they're more shareable than modal ones.
- Refusal rates only mean something paired with the seed, system prompt, and sampling temperature.
- The reviewer reading one transcript and the researcher reading a thousand are doing different jobs and should stop using the same verbs.
